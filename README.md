# DevPad
DevPad is a lightweight, browser-based Web IDE that lets users write, run, and preview HTML, CSS, and JavaScript in real time.
It runs entirely on the client side, making it fast, secure, and easy to use without any setup
Features:
 Live Code Editing for HTML, CSS, and JavaScript

 Real-Time Preview rendered instantly in the browser

 Tabbed Editors for smooth switching between languages

 Save & Load Projects using JSON files and LocalStorage

 Optional JavaScript Test Execution

 Keyboard Shortcuts

    Ctrl + Enter → Run code

    Ctrl + S → Save project

Fully Responsive Design (Mobile & Desktop supported)


Tech Stack : 
HTML5

CSS3 (Custom Tailwind-style design system)

JavaScript (ES6+)

Ace Editor

LocalStorage API
